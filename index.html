Color Scheme for vim

Follow easy steps to get a colorful vim developing environment.

First, pick your fav color scheme from HERE!! Then, add it to the following directory $ ~/.vim/colors/ If the directory is not exists, create it with following command $ mkdir -p ~/.vim/colors/

Finally, edit the .vimrc file in ~ $ vim ~/.vimrc 1 " FILE : .vimrc"
2 " DESCRIPTION : my vim envrionment setting "
3 syntax on
4 colorscheme COLOR_SCHEME_NAME
5 set t_Co=256
6 set tabstop=4
7 set shiftwidth=4
8 set number
9 set hlsearch
10 set autoindent
11 set ruler
12 set showmode
13 highlight Comment ctermfg=cyan
14 highlight Search term=reverse ctermbg=4 ctermfg=7
15 "Above is my vim environment setting"

Note:

  • first line for enabling the color
  • second line to select color scheme name, ex: COLOR_SCHEME_NAME = lucid (not need to add .vim)
  • third line to enable 256 colors.
  • forth line to set tab size = 4 space
  • fifth line to set auto indent size = 4 space
